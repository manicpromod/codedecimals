Hadoop

Hadoop is a open source framework to process and store large amount of data.Doug Cutting who is the founder of hadoop coined the word Hadoop after his kid's yellow elephant doll.

Hadoop Ecosystem

Below image shows the Hadoop Ecosystem
Hadoop ecosystem contains hdfs,YARN,MapReduce and other tools which will support the data analysis and processing.

HDFS

Hadoop provides HDFS (Hadoop Distributed File System) which is a framework for the data analysis and trasformation for large set of data. There are majorily four components including Namenode, DataNode, Job Tracker and Task Tracker.

NameNode

NameNode is a node which stores metadata of the the filesystem. It contains information such as which blocks are stored in which datanode,file maps to the block locations. The namenode maintains two in-memory tables, one which maps the blocks to datanodes (one block maps to 3 datanodes for a replication value of 3) and a datanode to block number mapping.

DataNode

Datanode is the node which stores the actual data. All datanode sends the heartbeat to the namenode for every 3 seconds.If the namenode does not receives any heartbeat from the datanode for more than 10 minutes then this data node is considered as dead and hence it initiates the replica of datanode to the other datanodes. Every data is replicated into 3 datanodes.

JobTracker

The primary responsibility of the job tracker is resource management.It manages all the task trackers by assigning the tasks,tracking the progress,fault tolerance etc.

TaskTracker

The task tracker has a simple function of following the orders of the job tracker and updating the job tracker with its progress status periodically.
